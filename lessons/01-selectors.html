1. CSS 선택자 (Selectors)

개념 설명

CSS 선택자는 어떤 HTML 요소에 스타일을 적용할지 결정하는 규칙입니다. 가장 기본적인 선택자로는 태그, 클래스(.), 아이디(#) 선택자가 있습니다.

예시 코드

HTML

<p>이것은 일반 p 태그입니다.</p>
<p class="highlight">이것은 클래스가 적용된 p 태그입니다.</p>
<p id="unique">이것은 아이디가 적용된 p 태그입니다.</p>

CSS

p {
    color: black;
}

.highlight {
    color: blue;
    font-weight: bold;
}

#unique {
    color: red;
    text-decoration: underline;
}
