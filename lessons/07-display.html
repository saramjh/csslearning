7. Display 속성

개념 설명

display 속성은 요소가 어떻게 보이고 다른 요소와 어떻게 상호작용하는지를 결정합니다.

  • block: 항상 새로운 줄에서 시작하고, 사용 가능한 전체 너비를 차지합니다. width, height, margin, padding 속성을 모두 사용할 수 있습니다. (예: <div>, <p>, <h1>)
  • inline: 새로운 줄에서 시작하지 않으며, 내용에 필요한 만큼의 너비만 차지합니다. width, height 속성을 적용할 수 없습니다. (예: <span>, <a>, <img>)
  • inline-block: inline처럼 다른 요소와 같은 줄에 배치될 수 있으면서, block처럼 width, height, margin, padding 속성을 모두 지정할 수 있습니다.
  • none: 요소를 화면에서 완전히 사라지게 만듭니다. 공간도 차지하지 않습니다.
