17. 크로스 브라우징 (Cross-Browser Compatibility)

개념 설명

크로스 브라우징은 모든 웹 브라우저(Chrome, Firefox, Safari, Edge 등)에서 웹 페이지가 의도된 대로 동일하게 보이고 기능하도록 만드는 기술적 접근 방법입니다. 브라우저마다 CSS 속성이나 HTML 태그를 해석하는 방식이 조금씩 다를 수 있기 때문에 중요합니다.

  • 벤더 프리픽스 (Vendor Prefixes): 특정 브라우저에서만 지원되는 실험적인 CSS 속성을 사용하기 위한 접두사입니다. (예: -webkit-, -moz-, -ms-)
  • Can I Use...: 특정 웹 기술(CSS 속성, HTML 태그 등)을 각종 브라우저에서 얼마나 지원하는지 확인할 수 있는 매우 유용한 웹사이트입니다.
  • CSS Reset/Normalize: 브라우저마다 다른 기본 스타일을 초기화하여 일관된 스타일링을 시작할 수 있도록 돕습니다.

벤더 프리픽스 예시

아래 박스는 일부 속성에 벤더 프리픽스가 적용되어, 구형 브라우저에서도 둥근 모서리와 텍스트 선택 방지 기능이 동작할 수 있도록 합니다.

Vendor Prefixes 
.example {
    -webkit-border-radius: 15px; /* Chrome, Safari */
    -moz-border-radius: 15px;    /* Firefox */
    border-radius: 15px;         /* Standard */
}
