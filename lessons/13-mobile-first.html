13. 모바일 우선 (Mobile-First) 디자인

개념 설명

모바일 우선 디자인은 웹사이트 CSS를 작성할 때, 가장 작은 화면(모바일)의 스타일을 기본으로 시작하는 접근 방식입니다. 그 후 미디어 쿼리의 min-width를 사용하여 화면이 커짐에 따라 레이아웃과 스타일을 추가하거나 변경합니다.

장점:

  • 모바일 사용자에게 최적화된 경험을 우선적으로 제공합니다.
  • 불필요한 CSS 재정의(override)를 줄여 코드가 더 깔끔하고 효율적이 됩니다.
  • 점진적 향상(Progressive Enhancement) 철학에 부합합니다.

코드 분석

위 예제의 CSS 코드는 모바일 스타일을 기본으로 작성하고, @media (min-width: ...)를 사용해 화면이 넓어질 때 스타일을 추가하는 방식으로 구성되어 있습니다. 이것이 모바일 우선 접근법의 핵심입니다.

/* 기본 (모바일) */
.mf-container {
    flex-direction: column;
}

/* 태블릿 이상 */
@media (min-width: 768px) {
    .mf-container {
        flex-direction: row;
    }
}
