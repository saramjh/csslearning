12. 미디어 쿼리 (Media Queries)

개념 설명

미디어 쿼리는 뷰포트의 너비나 높이, 기기의 방향 등 특정 조건에 따라 다른 CSS 스타일을 적용하는 방법입니다. 반응형 웹 디자인의 핵심 기술입니다.

@media (조건) { /* 적용할 스타일 */ }

  • max-width: 지정된 너비 이하일 때 스타일을 적용합니다. (주로 데스크탑 -> 모바일 순서로 디자인할 때 사용)
  • min-width: 지정된 너비 이상일 때 스타일을 적용합니다. (주로 모바일 우선 디자인에서 사용)
  • and, not, only: 여러 조건을 조합하거나 특정 조건을 제외할 수 있습니다.
