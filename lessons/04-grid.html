4. Grid

개념 설명

CSS Grid는 2차원(행과 열) 레이아웃 시스템을 제공하여, 복잡한 웹 페이지 구조를 쉽게 만들 수 있도록 도와줍니다.

  • grid-container: display: grid;를 적용하는 부모 요소입니다.
  • grid-item: 그리드 컨테이너의 자식 요소입니다.
  • grid-template-columns: 그리드의 열(column)의 크기와 개수를 정의합니다. (예: 1fr 1fr 1fr, 100px 200px auto)
  • grid-template-rows: 그리드의 행(row)의 크기와 개수를 정의합니다.
  • grid-gap (또는 gap): 그리드 아이템 사이의 간격을 설정합니다.
