16. calc() 함수

개념 설명

calc() 함수를 사용하면 CSS 속성 값 내에서 직접 계산을 수행할 수 있습니다. +, -, *, / 연산자를 사용할 수 있으며, 서로 다른 단위(예: %와 px)를 함께 계산할 수 있다는 점이 매우 강력합니다.

주의사항: +와 - 연산자 양 옆에는 반드시 공백이 있어야 합니다. (calc(100% - 50px)는 올바르지만, calc(100%-50px)는 틀립니다.)

실시간 미리보기

이 박스의 너비는 calc(100% - 40px) 입니다.
